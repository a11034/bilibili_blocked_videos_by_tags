# Bilibili Blocked Videos By Tags

对Bilibili.com的视频卡片元素，以标签、标题、时长、UP主名称、UP主UID 来判断匹配，添加屏蔽叠加层或隐藏视频。
修改原始项目的UI界面，以兼容不同的屏幕分辨率和浏览器缩放。
